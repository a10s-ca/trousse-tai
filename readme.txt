Le document "event.json" représente un événement de webdiffusion d'une pièce de théâtre sous forme de données structurées au format JSON-LD utilisant le vocabulaire Schema.org.

Les données décrivent un événement complètement fictif : une diffusion web d’une pièce appartenant à une tétralogie captée dans le cadre d’une représentation régulière, puis diffusée en ligne un an plus tard.

Les références à des personnes réelles, des organisations réelles,  des œuvres réelles et des critiques réelles sont ajoutées uniquement dans le but de faciliter la compréhension de l’exemple par le lecteur.
